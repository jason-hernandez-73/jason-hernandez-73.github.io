The Biologist Who Codes

      My background is in biology. Like all scientists, biologists have to use statistics, and Excel is not always sufficient. I first got involved with coding in order to use R, SAS, and similar tools; but Python can do a lot of those functions, too. At present, my goal is to stay focused on science, and use my data analytics skills in that field.

      This is my GitHub page, but not my first website. That one can be found at The Vagabond's Camp, which has been up since 1999. It started as a venue to try to counteract the Y2K hysteria; but when (surprise, surprise) nothing came of that, I expanded to other topics, including counteracting some of the Creationist foolishness that just doesn't die, and celebrating nature and travel. All these years, it has been in pure html. Now that I have been introduced to CSS and Bootstrap, I will be improving it visually.
